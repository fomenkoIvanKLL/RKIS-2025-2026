# TodoList Application

Консольное приложение для управления задачами с поддержкой нескольких пользователей.

## Функциональность

### Основные возможности
- Добавление, редактирование и удаление задач
- Установка статусов задач (NotStarted, InProgress, Completed, Postponed, Failed)
- Просмотр задач с различными фильтрами
- Система отмены/повтора действий (Undo/Redo)
- Сохранение данных в файлы

### Многопользовательский режим (практическая работа №11)
- Регистрация новых пользователей
- Авторизация по логину и паролю
- Хранение профилей в файле `data/profiles.csv`
- Отдельное хранение задач для каждого пользователя в файлах `data/todos_<UserId>.csv`
- Защита данных - каждый пользователь видит только свои задачи
- Возможность выхода из профиля командой `profile --out`

## Команды

### Управление задачами
- `add [--multiline|-m] <текст>` - добавить задачу
- `view [флаги]` - просмотр задач
- `read <номер>` - просмотр полной информации о задаче
- `setstatus <номер> <статус>` - изменить статус задачи
- `delete <номер>` - удалить задачу
- `update <номер> "<новый текст>"` - обновить текст задачи

### Управление профилем
- `profile` - показать информацию о текущем профиле
- `profile --out` или `profile -o` - выйти из текущего профиля

### Системные команды
- `undo` - отменить последнее действие
- `redo` - повторить отмененное действие
- `help` - показать справку
- `exit` - выйти из программы

### Флаги для команды view
- `--index, -i` - показывать индекс задачи
- `--status, -s` - показывать статус задачи
- `--update-date, -d` - показывать дату изменения
- `--all, -a` - показывать все данные
